## ✨ Fitur Utama
* **3D Tilt Interaction**: Kartu profil yang bergerak dinamis mengikuti pergerakan kursor (Parallax Effect).
* **Modern Loading Screen**: Animasi *pulse* inisial "MA" yang memberikan kesan aplikasi premium.
* **Amikom Branding**: Skema warna ungu eksklusif (`#6d28d9`) yang mencerminkan identitas kampus.
* **Glassmorphism UI**: Efek kaca transparan dengan blur latar belakang yang sangat estetik.
* **No Image Needed**: Seluruh aset visual dibuat murni menggunakan **CSS3**.

## 🛠️ Teknologi
Proyek ini dibuat menggunakan *Vanilla Web Stack* tanpa library eksternal agar tetap ringan:
- **HTML5** (Semantik)
- **CSS3** (Custom Properties, Keyframes, Backdrop-filter)
- **JavaScript** (Event Listeners & DOM Manipulation)
